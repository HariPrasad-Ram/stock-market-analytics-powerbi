# stock-market-analytics-powerbi
Power BI dashboard analyzing 9 merged stock datasets with KPIs, 52-week metrics, time-series charts, and interactive slicers for multi-period stock performance insights.


Project Overview

This project delivers an end-to-end Stock Market Analytics Dashboard built using Power BI, DAX, and Power Query.
Data from 9 different stocks was merged into one unified dataset to analyze market performance across multiple time periods including daily, monthly, quarterly, and yearly.

The dashboard includes KPIs, trend charts, volume analysis, 52-week statistics, and dynamic slicers for professional stock performance monitoring.
